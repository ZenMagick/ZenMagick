Geo IP service based on code from http://geoip.cvs.sourceforge.net/geoip/php/
Database as available at: http://www.maxmind.com/download/geoip/database/


Installation
============
1) Unzip this plugin into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Download/buy a GeoIP database and upload into the plugin folder
   (or configure with full filename)
   NOTE: The plugin requires the original filename to be used as different
         code is executed, depending on the file format.
4) Enjoy


Limitations
===========
This plugin has only been tested with the free database versions (GeoIP.dat and
GeoLiteCity.dat).
