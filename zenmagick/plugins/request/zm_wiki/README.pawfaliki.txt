
                     ** Pawfaliki v0.5.1 **

Pawfaliki is a wiki written entirely in 1 php file.

It is designed to be easily configured/hacked by anyone with a little
experience of php.

It is free & open source.

The Pawfaliki web site can be found at:
    http://www.pawfal.org/pawfaliki
		
The Pawfaliki project page on Sourceforge can be found at:
    http://www.sourceforge.net/projects/pawfaliki

REPORTING BUGS & FEATURE REQUESTS
=================================

Please report bugs by following the "Bugs" link on the Pawfaliki project 
page and filling out the form. Likewise, please submit feature requests 
by following the "RFE" (Request Feature Enhancement) link.

    http://sourceforge.net/projects/pawfaliki/				

You can also send mail to the Pawfaliki developers mailing list:

    pawfaliki-developers@lists.sourceforge.net

But please make sure that you also submit a bug/feature request report to 
the appropriate tracker, because information in email can sometimes get lost
in the general flood of messages.


MAILING LISTS
=============

A general mailing list for users of Pawfaliki is available. There is also a
more technical list for Pawfaliki developers.

Please email questions or comments to the lists at:

    pawfaliki-users@lists.sourceforge.net
    pawfaliki-developers@lists.sourceforge.net
				
To subscribe to either mailing list please use the appropriate admin page:

    http://lists.sourceforge.net/lists/listinfo/pawfaliki-users
    http://lists.sourceforge.net/lists/listinfo/pawfaliki-developers
