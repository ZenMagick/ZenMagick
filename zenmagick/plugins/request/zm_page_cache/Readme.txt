This is a ZenMagick plugin adding page caching support.


Installation
============
1) Unzip this plugin into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Enjoy


Usage
=====
Right now this is a 1:1 port of the ZenMagick core page caching code, so all configuration/
features are the same.

Exception: The default caching strategy callback function name changed to zm_page_cache_request_cacheable()
