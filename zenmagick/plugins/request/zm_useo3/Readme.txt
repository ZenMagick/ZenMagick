This is the ZenMagick plugin for Ultimate SEO 3.x support.


Installation
============
1) Download and unpack Ultimate SEO 3.x and unpack into your zen-cart
   installation as documented. DO NOT MODIFY ANY FILES!!
2) Unzip the plugin package into the zenmagick/plugins directory.
3) Install plugin using the ZenMagick Plugin Manager.
4) Reload a storefront page to activate Ultimate SEO installation itself
5) Configure Ultimate SEO 3.x options


After the plugin has been installed you should have the Ultimate SEO
configuration options available in the admin interface.
