This is the ZenMagick plugin for pretty link (SEO) support.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Add the provided .htaccess redirect rules to your main .htaccess file
   NOTE: If you have already a .htaccess file, make sure to not override
   existing rules.
