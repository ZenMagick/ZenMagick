This directory contains all the stuff that didn't find a place anywhere else.

* sql
  The sql directory contains all the SQL that ZenMagick needs for it's unique features.

* docs
  Release documentation

* external_example.php
  Some sample code that show how ZenMagick may be used from external PHP code.
  Note that there are a couple things to edit in order to make this work for your installation.

* legacy_settings.php
  Contains settings that got removed as they are no longer used or deemed unnecessary (usually 
  the layout related ones)
  Should your template still rely on those, copy this file into the zemnagick/core/settings 
  folder to add them again
