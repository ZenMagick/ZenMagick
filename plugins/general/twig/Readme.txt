Twig for ZenMagick


Installation
============
1) Unzip this plugin into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Enjoy


Twig
=======
.. is not included, but required in a twig subfolder. The plugin will attempt to load
Twig's Autoloader from: twig/lib/Twig/Autoloader.php
