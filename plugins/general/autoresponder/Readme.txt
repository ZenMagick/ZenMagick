Autoresponder - cron triggered emails
=====================================

This is the ZenMagick implementation of the zen-cart mod Autoresponder+,
http://www.zen-cart.com/index.php?main_page=product_contrib_info&cPath=40_60&products_id=1068.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Configure as required using the Plugin Manager

NOTE: Requires the cron plugin for scheduling.
