Flyout Categories
=================
This is a ZenMagick implementation of the zen-cart mod css_flyout_menu_1-3-8a,
http://www.zen-cart.com/index.php?main_page=product_contrib_info&products_id=75.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
