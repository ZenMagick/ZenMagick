This is a ZenMagick plugin adding page stats at the bottom of the page.

This is mainly for testing and debugging.


Installation
============
1) Unzip this plugin into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Enjoy


Additional configuration
========================
The setting 'plugins.pageStats.showEvents' can be set to true/false to override the plugin admin setting.
