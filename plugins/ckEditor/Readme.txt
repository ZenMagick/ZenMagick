CKEditor WYSIWYG plugin
=======================
Plugin to use the CKEditor - http://ckeditor.com/.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Download the CKEditor package from http://ckeditor.com/download .
3) unzip it into zenmagick/plugins/ckeditor/content.
4) Download the CKEditor jquery plugin from http://www.fyneworks.com/jquery/ckeditor/#tab-Download
   All that is requiredis the file 'jquery.CKEditor.pack.js'
5) Copy into zenmagick/plugins/ckeditor/content/ckeditor.
6) Install plugin using the ZenMagick Plugin Manager.

Once installed, CKEditor should appear in the list of available editors.

To make this the default editor, click on your username in the ZenMagick 
admin and choose it in the dropdown list of editors.
