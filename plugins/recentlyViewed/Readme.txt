Recently viewed sidebox
=======================
This is a ZenMagick implementation of the zen-cart recently viewed mod,
http://www.zen-cart.com/downloads.php?do=file&id=500.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Install plugin using the ZenMagick Plugin Manager.
3) Configure as required using the Plugin Manager
4) Run 'Create dummy files for all sideboxes' from the ZenMagick installation page (Configuration -> Installation)
5) Configure sidebox in zencart admin as usual

Custom template
===============
Copy the plugin sidebox template into your theme and start editing.
