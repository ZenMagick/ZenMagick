TinyMCE WYSIWYG plugin
======================
Plugin to use TinyMCE - http://tinymce.moxiecode.com/.


Installation
============
1) Unzip the plugin package into the zenmagick/plugins directory.
2) Download the jQuery package from http://www.tinymce.com/download/download.php .
3) unzip it into zenmagick/plugins/tinyMCE/content.
4) Install plugin using the ZenMagick Plugin Manager.

Once installed, TinyMCE should appear in the list of available editors.

To make this the default editor, click on your username in the ZenMagick 
admin and choose it in the dropdown list of editors.
